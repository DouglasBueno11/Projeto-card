HTML:

HTML:

HTML, que significa HyperText Markup Language, é a base para a criação de páginas da web. É como o esqueleto que fornece estrutura e organização ao conteúdo que você vê em um site.

Elementos:

Estas são as unidades básicas do HTML e definem as diferentes partes de uma página web. Os elementos são criados usando tags, que são escritas entre colchetes angulares( e ). Por exemplo, a h1 tag define um título e a p tag define um parágrafo. p

Tags:

Vêm em pares, uma tag de abertura e uma tag de fechamento, que definem o início e o fim de um elemento. O conteúdo que você deseja exibir fica entre as tags de abertura e fechamento.

Atributos:

São adições opcionais às tags que fornecem mais informações sobre um elemento. Por exemplo, a tag pode ter um srcatributo que especifica a origem da imagem.

CSS:

CSS:

CSS, que significa Cascading Style Sheets, trata da apresentação de uma página web. É como a pintura, as fontes e o layout que tornam um site visualmente atraente e fácil de usar.

Estilos:

Estas são as definições que controlam como os elementos HTML são exibidos. Eles incluem coisas como tamanho da fonte, cor, imagens de fundo e posicionamento.

Seletores:

Têm como alvo os elementos HTML específicos que você deseja estilizar. Os seletores podem direcionar elementos por ID, classe, nome de tag ou uma combinação destes.

Regras:

Combinam seletores e estilos para especificar como os elementos selecionados devem ser exibidos. Uma regra CSS normalmente consiste em um seletor seguido por um bloco de declaração que inclui os estilos que você deseja aplicar.
